HTML5 and Javascript CSV Parser

1. Actors

Add the actors. One actor on each line. If you need a static power for each actor, add square brackets with the value of the power after the actor name (ex Actor 1 [0.75] or USA[0.9]

2. Issues

Add the issues. The default range of an issues is between 0 and 100. If you need an other range, you can specify your range between square brackets separated with a comma, in example [0,100] or [2,6].

3. Data

Specify for each issue the position, salience and power of the actor.

The Salience and Power are ranged between 0 and 1. The position between the range of the issues.
